Design System Overview

Color Token Architecture

  • Light Mode: Clean White · Light Grey · Deep Navy · Teal / Gold
  • Dark Mode: Deep Navy · Dark Slate · Clean White · Electric Blue / Gold

Typography System

  • Headings: Playfair Display — elegant serif for titles
  • Body Copy: Inter — clean, modern sans-serif

Advanced JavaScript Features

  • dynamic-theme.js — Global Light/Dark toggle via CSS class switching and localStorage persistence
  • carousel.js — Modular photo & video carousel reused on Athletics and Gallery pages; nextSlide() / prevSlide() update the DOM dynamically

Micro-Interactions & Hover Effects

  • Sport cards: translateY lift + scale + drop-shadow on hover
  • Buttons & nav links: smooth accent color fill transitions
  • Hero ring pulse animation on home page
  • Staggered fade-up entrance animations sitewide
